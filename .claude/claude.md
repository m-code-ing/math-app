# Math App - Claude Configuration

## Project Overview
A math learning web app for children (ages 5-7) focused on basic addition and subtraction up to 100.

## Key Technologies
- React with TypeScript
- IndexedDB for local storage
- CSS for child-friendly UI
- No external APIs or databases

## Important Files
- `REQUIREMENTS.md` - Project requirements and specifications
- `app/src/` - Main application code
- `.claude/agents/` - Specialized AI agents for different aspects

## Development Guidelines
- Keep UI simple and child-friendly (ages 5-7)
- Use large buttons and bright colors
- Focus on visual learning aids
- All data stored locally in IndexedDB
- Offline-first design

## Current Status
- Requirements documented
- Ready to start React app setup and basic prototype development
